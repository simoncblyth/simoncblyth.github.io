Material Code Mapping Generalization¶

Persisted gensteps contain material indices, in order to map these to actual materials it is necessary to have a code to material name mapping.

env/chroma/G4DAEChroma¶

  • gdc-
  • ~/env/chroma/G4DAEChroma

NB the ChromaMaterialMap.json is an input to the generation of gensteps, the actual values of the codes are immaterial... so long as the map covers all materials and the codea are unique.
