SCRAPE¶

During scrape follow progress with:

exist-tail access

FUNCTIONS FOR DEV MACHINE¶

scrape-dir
cd to the data directory
scrape-wipe
delete the current scrape both in file system and database
scrape-status
compare time stampes on lastrestore and the tgz to ascertain what needs doin
scrape-update
perform the scrape after checking status and environment consistency redoing classification if needed CAUTION this can take 5hrs during which the machine will be mostly unusable for anything else due to illustrator windows coming and going
scrape-tgz
pack up the scrape into tgz
scrape-sync
copy the tgz across to production

FUNCTIONS FOR PRODUCTION MACHINE¶

scrape-explode
explode the tgz
