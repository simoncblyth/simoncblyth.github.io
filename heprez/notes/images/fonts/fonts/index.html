ILLUSTRATOR MATH FONTS¶

For these fonts to be available to illustrator you will need to restart it

The fonts are no longer available at the first URL or second where it redirects

  • http://www.esm.psu.edu/mac-tex/fonts/TeX-Illustrator.sit
  • http://mactex-wiki.tug.org/wiki/index.php?title=Fonts

http://www.ualberta.ca/~kbeach/eps_tips.html

A very good commercial alternative is the OpenType font UniMath from Applied Symbols. On MacOS X, the fonts should be installed by moving the Tex-Illustrator folder into the /Library/Application Support/Adobe/Fonts/Reqrd/ directory for versions up to CS2 and into /Library/Application Support/Adobe/Fonts/ for CS3.

These fonts are already installed

simon:fonts blyth$ ll "/Library/Application Support/Adobe/Fonts/Tex-Illustrator"
total 7512
drwxr-xr-x@ 140 blyth  staff  4760 11 Feb  2001 .
drwxrwxr-x    5 root   admin   170 19 Mar  2008 ..
-rwxr-xr-x@   1 blyth  staff     0 11 Feb  2001 CM
-rwxr-xr-x@   1 blyth  staff     0 11 Feb  2001 CMB10
-rwxr-xr-x@   1 blyth  staff     0 11 Feb  2001 CMBSY
-rwxr-xr-x@   1 blyth  staff     0 11 Feb  2001 CMBSY10
-rwxr-xr-x@   1 blyth  staff     0 11 Feb  2001 CMBSY5

Secure the .sit:

cd /usr/local/heprez/data/images/fonts
fonts-tex-illustrator-upload
=== htdocs-up : Tex-Illustrator.sit to C2R:/var/scm/tracs/heprez/htdocs/../Tex-Illustrator.sit

Verify:

( cd /tmp ; curl -O $(fonts-tex-illustrator-url) )
diff Tex-Illustrator.sit /tmp/Tex-Illustrator.sit
