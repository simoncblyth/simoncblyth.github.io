TEX2SVGPNGPDF¶

This is an combination of tex2svg- and svg2png- using dvisvgm and batik to do the conversions:

tex -- dvi -- svg
               |
               *---   png + pdf

Propagating a qtags latex change¶

Delete the ones you want to change:

b2mc:localhost heprez$ cd $HEPREZ_DATA/images/qtags
b2mc:qtags heprez$ l */*f*.*
