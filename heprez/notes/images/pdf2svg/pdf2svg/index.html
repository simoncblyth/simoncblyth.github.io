PDF2SVG¶

The PDF files were created with pdflatex from latex snippets. Illustrator is able to parse the PDF and generate an SVG with math fonts intact.:

tex -- pdf -- svg

As alternative approach is to use dvisvgm:

tex -- dvi -- svg

FUNCTIONS¶

pdf2svg-convert-fast dir

convert all .pdf found in dir/pdf/name.pdf into dir/svg/name.svg using applescript controlled illustrator CS A dynamic Makefile is generated to do fast timestamp checks between the source and targer files. This means that it completes in seconds when there are no conversions to do.

psd2svg-qtags/extras/pdgs/pdg

Invoke the converter in:

$HEPREZ_DATA/images/{qtags,extras,pdgs,pdg}

Distinction between “pdgs” and “pdg” needs clarification ... But I suspect that “pdg” needs deletion, see heprez:ticket:101 clarification needed on next updating the indices

But the stylesheets and xquery machinery expect /db/hfagc_system/pdgs.xml

svg2pdfpng-convert-dir srcd [tgtd]

convert all .svg in srcd into pdf and png in tgtd

using Adobe Illustrator. The target directory defaults to the src directory

if both png and pdf targets are newer than src, conversions are not performed

svg2pdfpng-convert-pipe

mdfind -onlyin $HEPREZ_DATA/images/qtags/ilcs_svg_glyphs_used ‘kMDItemFSContentChangeDate > $time.today(-10)’ | svg2pdfpng-convert-pipe
